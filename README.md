# Simple Counter & Color Changer App

## This project does: 
A fun web app featuring:

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Uncompress the .RAR file and open index.html in a modern web browser, or use 'Go Live' in VS Code
